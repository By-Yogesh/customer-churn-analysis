# Customer Churn Analysis

This project analyzes customer churn data to identify key factors
driving customer attrition and provide actionable business insights.

## Tools Used
- Python (pandas, matplotlib, seaborn)
- SQL
- Excel

## Project Structure
- data/: raw and processed datasets
- notebooks/: data cleaning, EDA, and churn driver analysis
- sql/: analytical SQL queries
- dashboard/: dashboards and visual outputs
